# angular starter kit
# author: Marcin So≈Çtysiuk
# version 1.0
# description: angular frontend development environment with sass
# requirements: nodejs, phantomjs, npm, bower, sass

1. Install
---------------------------------------
First please install nodejs, phantomjs, npm, bower, sass
then run cmd 'install.sh'

2. Run
---------------------------------------
How to run frontend env on localhost:9000 ?
run from cmd 'grunt server'

3. Build production app
---------------------------------------
run from cmd 'grunt build'
app is available from folder /target/app

4. Unit tests
---------------------------------------
run unit tests from cmd 'frontend-test.sh'

run coverage from cmd 'frontend-coverage.sh'
then we can open graph from coverage\PhantomJS 1.9.1 (Windows 7 0.0.0)\index.html

5. Developer environment:
---------------------------------------
open with sublime text editor file syngular.sublime-project
